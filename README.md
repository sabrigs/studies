# About

This repository contains books, courses, videos, and other resources that are part of my learning journey in software development.

The goal is to keep everything organized in a single place and track my progress over time.

I decided to keep these materials public because every programming journey is different. In the same way I learned from and felt inspired by other developers sharing their paths, I hope this repository can also help someone through the resources, notes, and references collected here.
